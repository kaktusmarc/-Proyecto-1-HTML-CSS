Estilos de texto


La siguiente lista presenta algunas etiquetas HTML básicas:

  • <p>: Define un párrafo de texto.
  • <h1>, <h2>, ..., <h6>: Definen encabezados de diferentes niveles.
  • <b>: Aplica negrita al texto.
  • <big>: Incrementa el tamaño del texto.
  • <super>: Renderiza el texto como superíndice.
  • <mark>: Resalta el texto con un color de fondo, similar al marcado de texto en amarillo.
  • <u> (Subrayado): Subraya el texto.
  • <div> (División): Define una sección o contenedor genérico que se utiliza para organizar y estructurar el contenido.
  • <span>: Similar a <div>, pero utilizado para aplicar estilos o manipulaciones específicas a porciones más pequeñas de texto.
  • <br> (Salto de línea): Crea un salto de línea dentro del texto.
  • <hr> (Línea horizontal): Inserta una línea horizontal para separar secciones.
