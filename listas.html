Listas


Listas Ordenadas (<ol> - Ordered List):

Crea una lista ordenada numerada o alfabética.


Listas No Ordenadas (<ul> - Unordered List):

Crea una lista sin un orden específico, utilizando viñetas o puntos.


Por otra parte, las listas anidadas en HTML son listas que están contenidas dentro de otras listas. Esto significa que dentro de un elemento de lista (<li>), puedes incluir otra lista, ya sea una lista ordenada (<ol>) o una lista no ordenada (<ul>), y a su vez, esta lista interna puede contener sus propios elementos de lista.
