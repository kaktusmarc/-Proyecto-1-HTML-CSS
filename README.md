# -Proyecto-1-HTML-CSS
Abrir desde index.html

La página web es un espacio dedicado a explicar distintos conceptos básicos de HTML y CSS. Incluye una sala con un slider de imágenes de zapatillas. Cuenta con un favicon y un cursor personalizado en forma de zapatilla que cambia al pasar sobre los botones. El logo de Nike funciona como acceso directo para volver al index. Además, incorpora animaciones tanto en el menú como en los botones.
