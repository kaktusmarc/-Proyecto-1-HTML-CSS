selectores y propiedades


Elementos: Seleccionan un tipo de elemento HTML (ejemplo: p para párrafos).

Clases: Seleccionan elementos con una clase específica (ejemplo: .clase).

IDs: Seleccionan un elemento con un ID único (ejemplo: #miId).

Propiedades Comunes:

  • Color: color: valor;
  • Tamaño de fuente: font-size: valor;
  • Margen y Relleno: margin: valor; y padding: valor;
  • Borde: border: valor;

Uso de Colores y Unidades:

  • Colores pueden ser nombres, códigos hexadecimales (#RRGGBB) o funciones (rgb()).
  • Unidades pueden ser pixeles (px), porcentajes (%), em (em), entre otras.
