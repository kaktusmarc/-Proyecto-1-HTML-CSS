tablas


  • <table>: Define una tabla en HTML.
  • <tr>: Define una fila dentro de una tabla.
  • <th>: Define una celda de encabezado en una fila o columna. Se utiliza para títulos y encabezados.
  • <td>: Define una celda de datos en una fila o columna. Se utiliza para el contenido principal de la tabla.
  • <thead>: Agrupa las filas que actúan como encabezados de la tabla.
  • <tfoot>: Agrupa las filas que contienen resúmenes o totales de los datos de la tabla.
  • <rowspan>: Propiedad utilizada dentro de <td> o <th> que especifica cuántas filas debería abarcar una celda en particular.
  • <colspan>: Propiedad utilizada dentro de <td> o <th> que especifica cuántas columnas debería abarcar una celda en particular.
