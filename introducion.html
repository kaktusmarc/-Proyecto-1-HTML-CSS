introducción a css


Definición y Propósito: CSS, o Cascading Style Sheets (Hojas de Estilo en Cascada), es un lenguaje de diseño que describe cómo se presenta un documento HTML. Su propósito es separar la estructura (HTML) de la presentación visual, permitiendo un diseño consistente y fácil de mantener.


Relación con HTML: Mientras HTML organiza y estructura el contenido de la página, CSS se encarga de aspectos visuales como colores, fuentes y diseño. Se vinculan mediante reglas de estilo que definen cómo se deben mostrar los elementos HTML.


Sintaxis Básica: Una regla CSS tiene la siguiente estructura: selector { propiedad: valor; }. El selector elige los elementos a los que se aplicará el estilo, y las propiedades y valores especifican cómo se verán.
